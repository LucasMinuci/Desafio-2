HTML: A Base da Web

HTML, ou HyperText Markup Language, é a linguagem fundamental para criar páginas web. Ele fornece a estrutura básica de um site através de elementos e tags que organizam e formatam o conteúdo. Com HTML, você pode definir títulos, parágrafos, listas, links, imagens e muito mais.

Cada página HTML é composta por uma série de tags, como (h1) para títulos, (p) para parágrafos e (a) para links. Essas tags são usadas para marcar diferentes partes do conteúdo e indicar como elas devem ser exibidas no navegador.

HTML é essencial para a construção de websites, pois é o ponto de partida para a criação de qualquer página na web. Ele trabalha em conjunto com outras tecnologias, como CSS (para estilo) e JavaScript (para interatividade), para criar experiências web ricas e dinâmicas.

Historia do HTML

O HTML foi inventado por Tim Berners-Lee, um cientista da computação britânico. A primeira versão do HTML foi desenvolvida em 1991. Tim Berners-Lee criou o HTML como parte de um projeto maior para desenvolver a World Wide Web, uma maneira de conectar e compartilhar informações de forma mais acessível através de uma rede de computadores

A ideia

A ideia de Berners-Lee era criar um sistema que permitisse a publicação e o compartilhamento de documentos de forma simples e organizada. O HTML, junto com o HTTP (HyperText Transfer Protocol) e o URL (Uniform Resource Locator), formou a base dessa nova forma de comunicação digital, que rapidamente se tornou fundamental para a estrutura da web como conhecemos hoje.

HTML na pratica

  1. A tag (font) é uma tag obsoleta usada para definir o estilo da fonte, como cor, tamanho e tipo. Em HTML5, seu uso é desencorajado em favor de CSS.

    < p >< font color="red" size="5">Texto em vermelho e maior.< /font>

  2. A tag (del) é usada para marcar texto que foi removido ou deletado de um documento.

    < p > O valor original era < del >$100< /del >, mas agora é < strong>$80 .< /p>

  3. A tag (p) define um parágrafo de texto.

    < p>Este é um parágrafo de exemplo. Ele é delimitado pelas tags <p> e </p>.< /p>

  4. A tag (abbr) é usada para definir uma abreviação ou acrônimo. Pode incluir o atributo title para fornecer a forma completa do termo.

    < p>O HTML significa < abbr title="HyperText Markup Language">HTML.
